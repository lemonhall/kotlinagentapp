# Terminal Command Catalog（目标：5000）

日期：2026-02-18
用途：汇总本次会话里讨论过的伪 CLI 命令面候选，便于后续逐条挑选入白名单（terminal_exec）。

## 约定

- 每行一条命令：<namespace> <verb>。
- 允许附带一句说明（— ...），但命令本体不带参数。
- 本文件里的所有条目均为候选命令面，不代表已实现。

---

## 已实现（v11）

- terminal_exec hello — 输出 HELLO ASCII + lemonhall 签名，并写审计 run。
